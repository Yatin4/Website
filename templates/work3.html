IIT Roorkee Research Internship

The aim of this project is to model the microstructural porosity observed in materials that are additively manufactured. The algorithm to create the FEM model representing porosity in the additively manufactured material is to be validated by carrying out tensile load simulations and comparing the results with experimental data for the same material. In particular, the material of interest in this project is the Aluminium alloy AlSi10Mg. The algorithm is realistic in the way it creates the model since the pores generated are at random locations and are non-intersecting, also providing the user with control on the randomness of the sizes of the pores. Moreover, the developed algorithm has been successfully validated by comparing the values of certain parameters (like yield strength and ultimate strength) from the results of tensile load simulations of the created model, with the same parameters discussed in a research study as a result of experimental tests conducted on the same material (AlSi10Mg). The values obtained from the simulation were extremely close to the ones mentioned in the research study (this is more than sufficient since heterogeneity would be different for each model), thereby validating the algorithm used to create the model. Furthermore, the validated algorithm was made to scale the model up to the specimen level and results showed consistency in the model’s heterogeneity as it scaled up.
